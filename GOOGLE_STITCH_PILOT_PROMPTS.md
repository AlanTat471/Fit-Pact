# Google Stitch Pilot Prompts

Use these exact prompts in Google Stitch for the first three high-impact screens.

## Prompt 1: Dashboard

Design a mobile-first Dashboard screen for a weight-loss coaching app.

Requirements:
- Keep existing data fields and labels unchanged.
- Add a premium hero section with subtle blue-purple gradient.
- Show a Stats grid with KPI cards (BMI, Body Fat %, Classification, Starting Weight, Weight to lose).
- Use clear hierarchy, spacing, and modern dark theme.
- Include subtle interaction style cues: hover/elevation/focus states.
- Accessibility: high contrast and clear text at small sizes.

## Prompt 2: Workouts (TDEE Calculator)

Design a mobile-first TDEE Calculator screen.

Requirements:
- Two major sections: Personal Information and Calculated Values.
- Maintain all existing field names and metric context.
- Use elevated card design, better spacing, and grouped input rhythm.
- Add an informative header block with concise subtitle.
- Keep visual style compatible with dark theme and bright fitness accents.
- Include states for loading, validation warning, and successful calculation.

## Prompt 3: Payment Details

Design a mobile-first payment plans screen.

Requirements:
- Keep Free, Weekly, and Fortnightly plan cards.
- Clearly emphasize active plan with stronger visual highlight.
- Preserve subscribe button labels exactly:
  - Subscribe - $4.99/billed weekly
  - Subscribe - $8.99/billed fortnightly
- Add trust indicators for security and payment protection.
- Style for strong conversion while keeping clarity and readability.
- Keep dialog visual language consistent with plan cards.
